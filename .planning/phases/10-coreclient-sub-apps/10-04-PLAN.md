```python
@app.command("purge")
def purge_url_cmd(
    ctx: typer.Context,
    url: str = typer.Argument(..., help="URL to purge from CDN cache"),
) -> None:
    """Purge a specific URL from CDN cache."""
    from bunny_cdn_sdk.cli._output import err_console, sdk_errors
    from bunny_cdn_sdk.core import CoreClient
    import typer as _typer
    state = cast("State", ctx.obj)
    if not state.api_key:
        err_console.print("Missing API key. Use --api-key or set BUNNY_API_KEY.")
        raise _typer.Exit(1)
    with sdk_errors():
        client = CoreClient(api_key=state.api_key)
        client.purge_url(url)
        _typer.echo(f"Purged: {url}")
```

| T-10-13 | Tampering | purge URL argument | accept | `CoreClient.purge_url()` URL-encodes the value via `urllib.parse.quote` before passing to the API; no server-side execution of the URL |
| T-10-14 | Information Disclosure | API key via purge command | accept | Standard env var / flag pattern; key is never echoed or logged |
