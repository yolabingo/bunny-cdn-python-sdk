# Plan 02-03: Public Re-exports

**Objective**

Implement `src/bunny_cdn_sdk/__init__.py` to re-export:
- `CoreClient` and `StorageClient` (forward references to Phase 3 implementations)
- All exception classes from `_exceptions.py`

Maintain alphabetical ordering to satisfy ruff's RUF022 rule.

**Purpose**

The public API of `bunny-cdn-sdk` is simple: `from bunny_cdn_sdk import CoreClient, StorageClient, BunnyAPIError`. All public imports go through `__init__.py`. This plan sets up those re-exports so Phase 3 can implement the service clients without worrying about `__init__.py`.

**Output**

- `src/bunny_cdn_sdk/__init__.py` with all public re-exports
- INFRA-10 requirement implemented
